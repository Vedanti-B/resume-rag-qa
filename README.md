# Resume Question Answering System (RAG)

This project answers resume-based questions using Retrieval-Augmented Generation.

## Features
- Resume PDF upload
- Section-aware chunking
- Semantic search using FAISS
- Answer generation using FLAN-T5
- Streamlit UI

## Run the App
```bash
pip install -r requirements.txt
streamlit run app.py
